手绘框 Demo

普通 HTML 内容保持不变，Rough.js 只负责在元素背后画 SVG 边框。 每个框固定 seed，所以刷新后线条不会乱跳。

单线柔和

单笔触，轻微歪斜，适合普通信息卡。

单线明显

单笔触，线条更厚，边缘更有手画感。

单线夸张

单笔触，高 roughness，适合重点模块。

双线柔和

Rough.js 默认多笔触，看起来像两条线重叠。

双线明显

多笔触更粗糙，也更有草稿味。

内外双框

一条外框加一条浅色内框，装饰感更强。

无填充线框

只画边，不画底色，适合覆盖在已有背景上。

斜线填充

保留手绘填充纹理，视觉存在感最强。

马克笔底色

边框加半透明浅色底，适合选中态和提示。
